Detects non-canonical variations of field names

The deb822 file format uses case-insensitive fields, which means that depends: foo and Depends: foo are treated as the same. However, every published field has a conventional way of being spelled, which provides consistency for reviewers and other developers. This inspection provides quick fixes to align the field name with the canonical variant.

This inspection generally covers case variants (e.g, the depends vs Depends example mentioned above) and also unnecessary X[SBC]- prefixes (e.g., XC-Package-Type vs. Package-Type).
