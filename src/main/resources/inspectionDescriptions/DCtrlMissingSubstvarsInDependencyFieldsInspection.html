Detects possibly missing substvars in dependency fields of package paragraphs in debian/control

The package paragraphs in debian/control should almost always have a dependency on a substitution variable like ${shlibs:Depends} for architecture dependent packages. This inspection reports some of these potentially missing substvars.

Limitations: Currently this inspection only covers the ${shlibs:Depends} substvars for now. The debhelper related substvars ${misc:Depends} and ${misc:Pre-Depends} are not covered yet.
