Detects when a field duplicates the value from the source paragraph.

In the debian/control file, the Package paragraphs inherits the value of fields from the Source paragraph where relevant. That implies that it is redundant to repeat the field in the Package if it contains the exact same value as the Source paragraph. This warning can serve to help you prune such redundant declaration or help you identify when a field incorrectly shares the same content as the Source paragraph.

Example:

Source: debhelper
Section: misc
Priority: optional
#...

Package: debhelper
# Overrides the previous Section value
Section: devel
# Unnecessary and just repeats the the above.  This would trip the warning.
Priority: optional
