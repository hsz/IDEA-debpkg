Detects non-standard Build-Profiles

Debian maintains a list of standardized Build-Profiles to be used in build dependency fields and the Build-Profiles field. In addition to these, any source package can declare custom build profiles of the format pkg.<source>.<name>, which are also considered valid by this inspection.

Please review https://wiki.debian.org/BuildProfileSpec for a description of what the Build-Profiles mean.

Limitation: Currently, only the Build-Profiles fields is checked. This will hopefully be fixed in a later version.
